Order on natural numbers. le is defined in Init/Peano.v as: 
Inductive le (n:nat) : nat -> Prop :=
  | le_n : n <= n
  | le_S : forall m:nat, n <= m -> n <= S m

where "n <= m" := (le n m) : nat_scope.

Require Export HoTT.Overture.
Require Export Coq.Init.Peano.

Local Open Scope nat_scope.

Implicit Types m n p : nat.

le is a pre-order

Reflexivity
Theorem le_refl : ∀ n, n ≤ n.
Proof.
  exact le_n.
Defined.

Transitivity
Theorem le_trans : ∀ n m p, n ≤ m → m ≤ p → n ≤ p.
Proof.
  induction 2; auto.
Defined.
Hint Resolve le_trans: arith v62.

Theorem le_Sn_0 : ∀ n, ¬ S n ≤ 0.
Proof.
  red; intros n H.
  change (match 0 with 0 ⇒ Empty | _ ⇒ True end);
  elim H; simpl; auto with arith.
Defined.

le is a order on nat

Antisymmetry


A different elimination principle for the order on natural numbers


Lemma le_elim_rel :
 ∀ P:nat → nat → Prop,
   (∀ p, P 0 p) →
   (∀ p (q:nat), p ≤ q → P p q → P (S p) (S q)) →
   ∀ n m, n ≤ m → P n m.
Proof.
  induction n; auto with arith.
  intros m Le.
  elim Le; auto with arith.
Defined.
