Tail-recursive plus

tail_plus is an alternative definition for plus which is tail-recursive, whereas plus is not. This can be useful when extracting programs.

Fixpoint tail_plus n m : nat :=
  match n with
    | O ⇒ m
    | S n ⇒ tail_plus n (S m)
  end.
